# Learnix - Learning Management System

A comprehensive learning management system built with Flutter, designed to help students and educators manage courses, lessons, and educational content.

## Features

- **Splash Screen**: Attractive startup experience
- **Login/Authentication**: Secure user authentication
- **Home Dashboard**: Personalized learning dashboard
- **Course Management**: Browse and manage courses
- **Lesson Content**: Interactive lesson viewing
- **Progress Tracking**: Monitor learning progress
- **Notifications**: Stay updated with important alerts
- **User Profile**: Manage personal information

## Screens

- Splash Screen
- Login Screen
- Home Screen
- Courses Screen
- Course Detail Screen
- Lesson Screen
- Notifications Screen
- Profile Screen

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Architecture

The app follows a clean architecture pattern with:
- State management using Flutter's built-in StatefulWidget
- Modular screen organization
- Consistent UI/UX design
- Responsive layout for different screen sizes

## Tech Stack

- Flutter (Dart)
- Material Design Components
- Responsive UI
